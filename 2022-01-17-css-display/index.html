이 글은 css 선택자 중 display none, block, inline, inline-block을 다룬다.

display: none

말 그대로 안 보이게 만든다.

하지만 visibillity: hidden과는 다르게 요소가 렌더링 되지 않는다.

따라서 영역을 차지하지 않는다.

영역을 차지 하지 않게 해야할 때 주로 사용한다.

display: block, display: inline

block은 inline과의 차이점을 중심으로 이해하는 게 좋다.

제일 중요한 건 block은 다음 태그가 다음줄에 나오게 하고

inline은 다음 태그에도 줄바꿈이 없다.

기본값이 block인 태그는 div 태그, p 태그, h 태그#, li 태그 등이 있다.

기본값이 inline인 태그는 span 태그, b 태그, i 태그, a 태그 등이 있다

inline 태그들을 기억하는 것이 좋은게 width와 height속성을 지정할 수 없으므로

width, height를 지정하려면 다른 태그를 생각해보거나 display 속성값을 바꿔주어야한다.

See the Pen Untitled by jiwonkim (@naamukim) on CodePen.

display: inline-block

inline-block은 줄바꿈이 되지 않지만 width와 height를 지정할 수 있다.

글 중간에 줄은 안 바꾸고 사용하고 싶을 때 사용하시면 된다.

(internet explorer 7이하에서는 지원되지 않는다…)

<style>
  .inline-block1 {
    display: inline-block;
    background: blue;
    height: 30px;
  }
  .inline-block2 {
    display: inline-block;
    width: 200px; /* 이 값은 이제 정상 작동합니다 */
    border: 1px solid #999;
  }
</style>
